--------------Installation du NDK-----------------------

Création d'une activité native C/C++
Tools -> SDK Manager > SDK Tools > cocher NDK, CMake

--------------Code boutons directions-----------------------------------

Le code initialise 1 Listener pour chaque bouton, qui lors du click vont initialiser une variable String ("direction"), et lancer la fonction C++ avec le nom du bouton correspondant en paramètre.

La fonction C++ récupère la String en paramètre, la convertie en un format String compatible C++ (const char*), puis compare cette String avec le texte du bouton grace à "strcmp()".
Si il s'agit du bon bouton, le texte en japonais est renvoyé sous format String compatible kotlin, et copié dans la variable de départ ("direction").

Enfin, la zone de texte prend la variable direction pour valeur.
(J'avais utilisé un widget editText plutôt qu'un textView au début, d'où la difficulté à editer sa valeur)

--------------Code boutons READ/WRITE-----------------------------------

On initialise la aussi 1 listener par bouton, au click on vérifie que le champ de texte ne soit pas vide et on lance la fonction callMultiplication avec le nom du bouton et la valeur du champ editText en paramètre (Valeur convertie de Editable vers int au prealable).

La fonction C++ récupère ces paramètres et verifie que le chiffre sois bien entre 1 et 10.
Si oui elle verifie de quel bouton il s'agit, réalise le calcul et renvoi le résultat dans une string. 
Sinon elle renvoi un message d'erreur dans une string.
